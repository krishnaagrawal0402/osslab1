#include <math.h>
#include <stdio.h>

// 1. Addition
void calcAddition() {
    printf("\nEnter two numbers: \n");
    int a, b;
    scanf("%d%d", &a, &b);
    int result = a + b;
    printf("The sum is %d\n", result);
}

}

//Sine
void calcSine() {
    printf("\nEnter the angle in degrees: ");
    int angle;
    scanf("%d", &angle);

    // Convert degrees to radians
    double radians = angle / 180.0 * 3.14;
    double answer = sin(radians);

    printf("The sine value is %f", answer);
}

Working-
Addition (calcAddition): Adds two integers and prints the sum.
Subtraction (calcSubtraction): Subtracts the second integer from the first and prints the result.
Multiplication (calcMultiplication): Multiplies two integers and prints the product.
Division (calcDivision): Divides the first integer by the second (integer division) and prints the quotient.
Exponentiation (calcexpo): Raises the first integer to the power of the second using pow, prints the result as an integer (may be inaccurate).
Sine (calcSine): Converts an angle from degrees to radians, computes the sine, and prints the result.
Cosine (calcCosine): Converts an angle from degrees to radians, computes the cosine, and prints the result.
Tangent (calcTan): Converts an angle from degrees to radians, computes the tangent, and prints the result.
Floor (calcFloor): Converts an angle from degrees to radians, computes the floor of the radians, and prints the result.
Ceiling (calcCeil): Converts an angle from degrees to radians, computes the ceiling of the radians, and prints the result.
Round (calcRound): Converts an angle from degrees to radians, rounds the radians to the nearest integer, and prints the result.
Absolute Value (calcAbs): Converts an angle from degrees to radians, computes the absolute value of the radians, and prints the result. 
Note: abs is for integers; fabs is better for floating-point numbers.
